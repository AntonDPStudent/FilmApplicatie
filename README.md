Project voor het vak Levenslang Leren

Filmapplicatie die draait in Azure en data uit Azure blob storage haalt. 

Doel: Aanleren van Javascript/react (latere integratie) en leren werken met de microsoft Azure omgeving.

Gebruikte technologieeen: 
- HTML
- CSS
- Javascript
- React (latere fase)
- Azure Blob storage
- Azure Static web apps
- Azure Storage accounts
